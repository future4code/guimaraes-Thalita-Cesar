# Astromatch <3

## Descrição do Projeto: 
O objetivo é fazer um clone do tinder. O projeto terá duas telas, uma com os profiles para dar match e outra com a lista de matchs e a opção de limpar matchs.

## Técnologias utilizadas:
* react
* bootstrap
* styled components
* postman
* react icons
